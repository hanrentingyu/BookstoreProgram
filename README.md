C++ 图书库存管理系统
一个基于 C++ 的轻量级图书库存管理工具，专为编程初学者和计算机科学学生设计。该项目展示了现代 C++ 的核心特性，包括面向对象编程、运算符重载、异常处理以及文件流操作。
适用场景：
● 计算机专业 C++ 课程大作业/实验报告
● C++ 语法练习（类、STL、文件操作）
● 个人库存管理小工具



主要功能
● 库存智能合并：自动识别相同 ISBN 的书籍，支持多次进货累加，并根据总价值和总数量自动计算加权平均价格。
● 严格数据验证：
○ 内置 ISBN-10 标准校验算法（模数 11 校验），确保图书编号合法。
○ 业务逻辑检查（如：库存为 0 时价格必须为 0）。
● 数据持久化：支持从文本文件读取库存，并将更新后的结果保存回文件，实现数据的长期存储。
● 异常安全机制：使用 try-catch 块处理非法输入和文件错误，防止程序意外崩溃。



技术栈与亮点
● 语言标准：C++17 (兼容 C++11)
● 核心组件：
○ Book 类封装：高内聚低耦合的设计。
○ std::map：使用红黑树管理库存，保证 ISBN 索引的高效查找。
○ 运算符重载：实现了 += 和 +，使库存合并逻辑像数学公式一样直观。
○ 友元函数：重载 >> 和 <<，实现流式输入输出。



项目结构

.
├── book.hpp        # 头文件：Book 类定义
├── book.cpp        # 实现文件：Book 类逻辑及 ISBN 校验
├── main.cpp        # 主程序：文件读写与业务逻辑
├── input.txt       # [输入] 进货数据文件
├── output.txt      # [输出] 最终库存文件
└── README.md       # 项目说明文档




快速开始
编译项目
使用支持 C++17 的编译器（如 g++）：

g++ -std=c++17 -O2 -Wall -o inventory main.cpp book.cpp

准备数据
在项目目录下创建 input.txt，每行代表一种新书的进货信息（格式：ISBN 数量 价格）：

0306406152 5 59.90
123456789X 10 25.00

运行程序

./inventory

程序运行后，控制台将显示处理日志，更新后的库存将保存至 output.txt。



输入输出格式说明
● ISBN：10 位字符串，最后一位可为 'X'。
● 数量：非负整数。
● 价格：浮点数（程序输出时保留两位小数）。
输出示例 (output.txt)：

0306406152 5 59.90
123456789X 10 25.00



核心算法逻辑
加权平均价格：当合并同 ISBN 书籍时，根据总价值与总数量自动计算新的加权平均单价。
ISBN-10 校验：内置标准的 ISO 2108 模数 11 校验算法，确保图书编号的合法性。
